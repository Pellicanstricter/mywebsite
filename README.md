# Nathaniel Steinrueck - Portfolio Website

A modern, responsive portfolio website to showcase your projects and skills.

## Project Structure

```
portfolio-website/
├── index.html          # Main HTML file
├── css/
│   └── style.css      # Stylesheet
├── js/
│   └── main.js        # JavaScript functionality
├── images/            # Place your images here
├── projects/          # Project-specific files
└── README.md          # This file
```

## Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Modern, clean UI with hover effects
- Sections for:
  - Hero/Introduction
  - About Me
  - Projects showcase
  - Skills
  - Contact information

## Getting Started

1. Open `index.html` in your browser to view the site locally
2. Update the content in `index.html` with your information
3. Add your images to the `images/` folder
4. Customize colors and styling in `css/style.css`
5. Add any additional JavaScript functionality in `js/main.js`

## Customization Tips

### Colors
The CSS uses CSS variables for easy color customization. Update these in `css/style.css`:
- `--primary-color`: Main brand color
- `--secondary-color`: Secondary brand color
- `--text-color`: Main text color
- `--bg-color`: Background color
- `--light-bg`: Light background for cards

### Content
Replace placeholder content in `index.html`:
- Update bio in the About section
- Add your actual projects in the Projects section
- List your skills in the Skills section
- Update contact links with your email, GitHub, LinkedIn, etc.

## Deployment

To deploy to https://nathanielsteinrueck.com/:
1. Choose a hosting service (GitHub Pages, Netlify, Vercel, etc.)
2. Upload all files maintaining the folder structure
3. Configure your domain to point to the hosting service

## Technologies Used

- HTML5
- CSS3 (with CSS Variables and Grid/Flexbox)
- Vanilla JavaScript
- No external dependencies required
